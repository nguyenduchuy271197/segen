# Tính năng chính

1. Chế độ View

- Hiển thị nội dung đã định dạng
- Vô hiệu hóa tất cả tương tác chỉnh sửa
- Cho phép copy text

2. Chế độ Edit

- Đầy đủ các công cụ định dạng text
- Hỗ trợ undo/redo
- Tự động lưu khi có thay đổi

3. Định dạng hỗ trợ

- Text styles (bold, italic, underline)
- Headings (H1, H2, H3)
- Lists (bullet và numbered)
- Blockquotes
- Links
- Images
- Code blocks
